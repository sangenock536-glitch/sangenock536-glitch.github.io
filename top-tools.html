  • Great template library

Anki — The Science-Backed Gold Standard

Anki uses a proven spaced repetition algorithm to show you cards at the optimal moment — just before you'd forget them. Used by medical students worldwide.

✅ Pros

  • Best SRS algorithm available
  • Completely free on desktop
  • Huge shared deck library
  • Supports images, audio, LaTeX

❌ Cons

  • Dated UI design
  • iOS app costs $24.99
  • Steeper initial setup
